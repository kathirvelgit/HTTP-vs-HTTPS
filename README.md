# 🌐 HTTP vs HTTPS

This repository contains notes on the key differences between **HTTP** and **HTTPS**.

## 📡 HTTP (HyperText Transfer Protocol)

- 📄 Data is sent in plain text.
- 👀 Anyone intercepting the connection may be able to view the transmitted information.
- ⚠️ Less secure compared to HTTPS.
- 🔓 Hackers can intercept and read the data.
- 🌍 URL starts with `http://`.
- 🔌 Uses port **80** by default.

## 🔒 HTTPS (HyperText Transfer Protocol Secure)

- 🔐 Data is encrypted using SSL/TLS.
- 🛡️ Provides secure communication between client and server.
- 💳 Protects sensitive information such as passwords, banking details, and personal data.
- ✅ Much more secure than HTTP.
- 🌍 URL starts with `https://`.
- 🔌 Uses port **443** by default.

## 📊 Key Differences and Easy Understandable

| Feature | HTTP | HTTPS |
|----------|----------|----------|
| Security | ⚠️ Less Secure | 🛡️ More Secure |
| Encryption | ❌ No Encryption | 🔐 SSL/TLS Encryption |
| Data Protection | 📄 Data sent in plain text | 🔒 Data sent in encrypted form |
| URL Prefix | `http://` | `https://` |
| Default Port | 80 | 443 |
| SSL Certificate | ❌ Not Required | ✅ Required |
